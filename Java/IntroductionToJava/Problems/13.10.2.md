# 13.10.2

30명의 승객을 싣고 가던 배가 폭풍우를 만나 배를 구하기 위하여 승객의 절반이 희생되어야 하는 상황이 되었다. 
따라서 승객 30명을 원형으로 세우고 어떤 일정한 지점부터 9번째에 해당하는 사람이 희생하기로 정책을 세웠다.
돌아가면서 매 9번째 사람을 제거하고 15명이 남을 때까지 반복한다. 살아남는 15개의 바람직한 위치를 구하는 프로그램을 작성하시오.
